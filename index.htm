Page Features

I am using "div" and "class" to define which sections of the page are styled in certain ways. As oppose to styling each element the same way this allows me to have them styled by groups instead.

I am also using icons on the headings which I have implemented using the 'i class' element. This allows me to have a visual representation of the contents included in each section.

I have also implemented a list and a table, both of which are skills I learned on my course. These can be seen in the "HTML and CSS" section, and in the section titled "What are my goals for 2025?.

At the top of the page I have added a section that links to different parts of the portfolio to help make it easier to get to where you want to go. As more content is added to the page, those links wwill become more and more useful.
